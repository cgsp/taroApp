## 第一个 taro 应用

##  项目目录
```bash
 taro-app
     ├─ config                # 配置目录
     │   ├─ dev.js               # 开发配置
     │   ├─ index.js             # 默认配置
     │   ├─ dev.js               # 生成配置
     ├─ dist                  # 编译结果目录
     ├─ src                   # 源代码
     │   ├─ actions              # redux
     │   ├─ constants            # redux
     │   ├─ pages                # 页面目录
     │   ├─ reducers             # redux
     │   ├─ store                # redux
     │   ├─ app.jsx              # 入口jsx
     │   ├─ app.scss             # 入口scss
     │   ├─ index.html           # h5页面，需要
     ├─ .editorconfig         # 磨平操作系统的差异配置
     ├─ .eslintrc             # 代码风格检测
     ├─ .gitignore            # git忽略提交配置
     ├─ package.json
     ├─ project.config.json   # 项目配置--微信小程序专用的
     ├─ README.md
     ├─ yarn.lock             # npm包锁定版本
```
